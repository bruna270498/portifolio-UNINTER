Aqui estão alguns dos meus projetos:

Tryunfo

O Tryunfo é um jogo de cartas inspirado no famoso jogo de tabuleiro "Super Trunfo". O objetivo do jogo é vencer o adversário utilizando cartas com atributos variados. O projeto foi desenvolvido em React e utiliza a biblioteca React Router para navegação entre páginas.

Clique aqui para ver o repositório do projeto

TrybeTunes

O TrybeTunes é uma aplicação que permite pesquisar e ouvir músicas de artistas. O projeto foi desenvolvido em React e utiliza a API do iTunes para buscar informações sobre as músicas. A aplicação possui funcionalidades como busca por artista, exibição de detalhes da música e adição de músicas à lista de favoritos.

Clique aqui para ver o repositório do projeto

TrybeWallet

O TrybeWallet é uma aplicação de controle financeiro que permite ao usuário registrar suas despesas em diferentes moedas. O projeto foi desenvolvido em React e utiliza a API da CoinGecko para obter as taxas de câmbio. A aplicação possui funcionalidades como adição, edição e exclusão de despesas, além de exibir o total gasto em reais.

Clique aqui para ver o repositório do projeto

React Testing Library

O React Testing Library é uma biblioteca de testes para aplicações React. O projeto foi desenvolvido com o objetivo de facilitar a escrita de testes unitários e de integração, garantindo que os componentes funcionem corretamente. A biblioteca fornece utilitários para renderizar componentes, simular eventos e verificar o comportamento esperado.

Clique aqui para ver o repositório do projeto
